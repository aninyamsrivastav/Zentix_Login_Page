# 🔐 Zentix Login Page

A simple login page built using **Python (Flask)**, **HTML**, and **CSS**.

## 🚀 Features

- 🔧 Flask-based backend
- 🎨 Responsive front-end with custom CSS
- 🧭 Basic login and dashboard routing
